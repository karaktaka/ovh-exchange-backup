# Prerequisites

## Create API Key

You need an API Key which can be created here: <https://api.ovh.com/createToken/index.cgi?GET=/*&PUT=/*&POST=/*&DELETE=/*>

And it should have the following access rights:
* "GET", "/email/*"
* "POST". "/email/*"
* "DELETE". "/email/*"
